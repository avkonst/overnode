Remove existing target peer node.
```md
> overnode [--debug] [--no-color] forget [OPTION] ... HOST ...

  Options:   Description:
  ----------------------------------------------------------------------------
  HOST       Hostnames or IP addresses of target peer nodes.
  ----------------------------------------------------------------------------
  -h|--help  Print this help.
  ----------------------------------------------------------------------------
```
