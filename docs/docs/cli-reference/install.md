Install overnode and the required dependencies.
```md
> overnode [--debug] [--no-color] install [OPTION] ...

  Options:   Description:
  ----------------------------------------------------------------------------
  -f|--force Force to re-install, if already installed.
  ----------------------------------------------------------------------------
  -h|--help  Print this help.
  ----------------------------------------------------------------------------
```
