Lookup DNS entries of a cluster.
```md
> overnode [--debug] [--no-color] dns-lookup [OPTION] ... HOSTNAME

  Options:   Description:
  ----------------------------------------------------------------------------
  HOSTNAME   Name to look up in the DNS register.
  ----------------------------------------------------------------------------
  -h|--help  Print this help.
  ----------------------------------------------------------------------------
```
