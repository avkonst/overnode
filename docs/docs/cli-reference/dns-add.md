Add extra DNS entries.
```md
> overnode [--debug] [--no-color] dns-add --ips IP,... --name FQDN [OPTION] ...

  Options:   Description:
  ----------------------------------------------------------------------------
  --name FQDN
             Fully qualified domain name of the DNS entry.
  --ips IP,...
             Comma separated list of IP address to update.
  ----------------------------------------------------------------------------
  -h|--help  Print this help.
  ----------------------------------------------------------------------------
```
