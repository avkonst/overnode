Add an additional target peer node to connect to.
```md
> overnode [--debug] [--no-color] connect [OPTION] ... HOST ...

  Options:   Description:
  ----------------------------------------------------------------------------
  HOST       Hostnames or IP addresses of target peer nodes.
  --replace  Forget all existing target peer nodes.
  ----------------------------------------------------------------------------
  -h|--help  Print this help.
  ----------------------------------------------------------------------------
```
