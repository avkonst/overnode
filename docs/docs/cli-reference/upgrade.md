Download and install newer version of overnode and dependencies.
```md
> overnode [--debug] [--no-color] upgrade [OPTION] ...

  Options:   Description:
  ----------------------------------------------------------------------------
  --version VERSION
             Specific version to upgrade to. Default is latest available.
             See available version online:
             https://github.com/overnode-org/overnode/releases
  ----------------------------------------------------------------------------
  -h|--help  Print this help.
  ----------------------------------------------------------------------------
```
